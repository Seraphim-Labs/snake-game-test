# snake-game-test

A tiny vanilla-JS Snake game used as a system test for the repo + Netlify automation.

## Controls
- Arrow keys or WASD — move
- Space — pause
- R — restart

## Stack
Single `index.html` + `styles.css` + `game.js`. No build step.

## Deploy
Static site on Netlify. Publish directory is the repo root, no build command.
